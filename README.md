# 🎤 Voice To Text Android App

An Android application that converts spoken words into text using the Android SpeechRecognizer API.  
The app provides a clean interface and supports word limits to help users capture speech accurately.

---

## 📱 Features

- 🎙 Real-time voice recognition
- 📝 Converts speech directly into text
- 🔢 Word limit support
- 🎨 Clean and simple user interface
- ⚡ Fast speech processing using Android SpeechRecognizer API

---

## 🛠 Tech Stack

- **Language:** Kotlin  
- **Platform:** Android  
- **API:** Android SpeechRecognizer API  
- **Build Tool:** Gradle  
- **IDE:** Android Studio
